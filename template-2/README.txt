# SereneStay Hotel Template Two

This is a second complete Bootstrap 5 hotel website template using the same
Serene Earth & Champagne color palette as the first template, but with a
completely different editorial luxury design.

## Pages included
- index.html
- rooms.html
- experiences.html
- dining.html
- spa.html
- gym.html
- events-halls.html
- contact.html
- booking.html

## Design differences from the first template
- Pill-shaped glass navbar instead of full-width transparent navbar
- Split editorial video hero with booking card
- Overlapping booking strip
- Asymmetrical editorial image sections
- Wide horizontal room cards
- Mosaic-style gallery support
- Fixed image luxury bands
- Split CTA sections
- Different page composition and layout flow

## Interactive features
- Sticky glass navbar on scroll
- JavaScript-controlled background color alternation
- Room detail modals on rooms page
- Booking tabs for Rooms and Halls & Events
- Automatic room stay calculation
- Simulated room availability results
- Booking summary step
- Event duration calculation
- Book and Pay placeholder buttons
